# List Styling Engine

Transform SharePoint lists and document libraries into fully art-directed views using column formatting, view formatting, row templates, and tile layouts. Works with any style token file in this collection to produce "I can't believe that's SharePoint" results.

![preview](./assets/preview.png)

## What you get

- A step-by-step workflow for mapping your list's columns to a style token file
- Complete `rowFormatter` JSON that replaces the default column grid with a fully designed card layout
- Per-column formatters for status badges, progress bars, and deadline compositions
- Instructions for creating a custom view so the formatting never touches your default "All Items" view

## When to use

Ask Copilot:

- *"style this list"* / *"style this library"* / *"theme this list"*
- *"apply the {neobrutalism|bento|pastel|terminal|…} style to this library"*
- *"visually transform this list"* / *"art-direct this view"*
- *"make this list look like {style}"*

Works against any list or document library. Pair it with one of the [style-*](../) token skills (e.g. [style-neobrutalism](../style-neobrutalism/), [style-bento](../style-bento/), [style-pastel](../style-pastel/)) to drive the visual language.

## Demo content

Sample files and the recommended column setup (`Status` / `Progress` / `Deadline`) for an end-to-end demo are in the [demo/](./demo/) subfolder. Includes 10 short Word documents on SharePoint Skills topics so the styled view has enough rows to look realistic. Skip this folder when uploading the skill to SharePoint.
